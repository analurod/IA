# 🤖 Artificial Intelligence (AI)

## 📌 Overview
This repository contains projects and experiments developed in the field of **Artificial Intelligence (AI)**, focusing on intelligent agents and problem-solving in simulated environments.

Currently, the repository includes the implementation of the **Wumpus World** and an **AI agent** capable of interacting with the environment, interpreting perceptions, and making autonomous decisions.

---

## 🔑 Features
- 📌 **Wumpus World Simulation**: Environment modeled as a grid with hazards and objectives.
- 📌 **Intelligent Agent**: Agent capable of making decisions based on perceptions.
- 📌 **Autonomous Navigation**: Exploration strategies within the environment.

---

## 🛠️ Technologies Used
- **Python** (main language)
- **NumPy** (environment modeling)
- **Random** (movement and simulation logic)
